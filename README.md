Panel de control para administrar y configurar facilmente [El launcher de Minecraft de Miguelki](https://github.com/miguelkix30/MiguelkiNetworkMCLauncher):
 ---
### **<ins><p align="center">Condiciones de uso</p>**
Si deseas utilizar el código de este fork deberás:
- Dar crédito de este repositorio
- Bifurcar el repositorio. No puedes descargar el código y subirlo a tu repositorio.

### **<ins><p align="center">AVISO IMPORTANTE</p>**
No se dará ningún tipo de soporte a cualquier repositorio bifurcado de este.

Este código está especificamente creado para [el launcher de Minecraft de Miguelki](https://github.com/miguelkix30/MiguelkiNetworkMCLauncher). Deberás ajustar alguna cosa si quieres utilizarlo para tu launcher.

### **<ins><p align="center">Como utilizar:</p>**
1- Subir los archivos del repositorio al servidor web
2- Modificar el archivo users.json con el usuario y contraseña para acceder. 
Por defecto: 
Usuario: Admin
Contraseña: 1234

**MUY IMPORTANTE**
Deberás proteger tu mismo el archivo users.json para que no pueda ser accedido desde fuera. No se incluye ningún archivo .htaccess que lo haga.
